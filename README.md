# 图书跟踪应用

这是一个使用React构建的书架应用，能够选择和归类你阅读过的图书、正在阅读的图书以及想要阅读的图书。

主页面还有一个“搜索”链接，该链接是一个搜索页面，使你能够查找图书并将其添加到书库中。当你从搜索页面返回主页面时，可以立即看到在书库的搜索页面所做的所有选择。
